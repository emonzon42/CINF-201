Practice Validation Assignment

There are numerous errors in the HTML on this page and they can appear anywhere in the code. If you right-click the page in Google Chrome and select View page source, the HTML for this page will appear.

Your job is to get this HTML document to validate. To do so, you must to go the W3 Validator, copy/paste your code, and then hit check. If you see only a green bar, then you have completed the assignment. A red/yellow bar indicates that work still needs to be done before the page meets good validation standards.

It is important to note that no content should be deleted. There is also no need for you to alter the content in the body tag to make it your own. Tags should only be added, deleted, altered, or moved around to pass validation. Don't worry about styles as the primary focus of this assignment should be on code validation.

To submit the assignment, you will upload the corrected file to your UAlbany web server area via FileZilla. The link should look like this: albany.edu/~username/cinf201/validation-exercise.html. Username will be replaced with your personal UAlbany ID.
