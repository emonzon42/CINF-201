Challenge 1

The form below should take in 3 inputs: height, width, and background color. When the user clicks the "Submit Box Info" button, the div below this form with an id of "box" should get the specified height, width, and background color added to it. You'll need to get the value of each input tag to use for styling empty div. You can use the setAttribute method on the div with an id of "box" to get these results. You can also use .style.width, .style.height, etc.

Challenge 4

There is an empty unordered list below this paragraph. Create an array with at least 4 animals in it. Using a for or while loop, have each item in the array printed onto the page inside of the list (innerHTML combined with "+=" and li tags).

    Challenge 5

    The following form takes in 3 inputs: width, length, and height. After a user clicks "Submit Dimensions", the form should tell the user the volume of their box. You will need to use a click event on the button and then grab the input values afterward. The formula for the volume of a 3-dimensional box is length x width x height. This is the same as the boxVolume challenge from last week but we are taking the inputs from a form instead of providing them ourselves.

    Example output: "The volume of your box is 27000."

    Challenge 6

    The div below is currently visible on the page. When the user clicks the "Toggle Div" button below, it should either hide or show the div depending on whether or not it is hidden already. For this, you'll need to use the .style property on the div to either make it display: none or display: block. You will also use an if/else statement to check the display property. When you check the display property, you should check if the value is equal to "block" or an empty string ("").
