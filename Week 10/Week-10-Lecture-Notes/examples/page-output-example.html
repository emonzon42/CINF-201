Creating Page Output

On this page, there are a few examples which demonstrate how to use JS to add some basic interactivity to your page. Some of the examples will create items on the page while others will alter already existing items. The code for each example is located in the script tags directly the HTML for that example. This is an example of internally placed JS. To view this code, right-click the page and select "View Source." There are accompanying comments to help explain what's happening in the code.

Adding Text to the Page

In this example, we will use the click of a button to add a paragraph tag to an empty div. When you click the button below, a paragraph will appear on the page.

To add HTML to a page, you need to use the "innerHTML" property on an element. For our code below, we use innerHTML on the div with an id of "emptyDiv." When you use innerHTML, you can place HTML tags inside of the quotation marks and it will display on the page as HTML.

Altering the Styles of an Element (First Example)

In this first example, we will use a button click to hide a paragraph. To do this we will target the paragraph's "style" property and use it to change the display to "none."

This paragraph will be hidden.

Altering the Styles of an Element (Second Example)

In this second example, we will use a button click to change multiple styles of an element at the same time (background color and text color of a paragraph). Like we did in the first example, we will attach an onclick event to the button.

Once the button is clicked, we will target the paragraph below by its id value and then alter its CSS styles. The difference in this example is that we are using the setAttribute method. With setAttribute, we can set any attribute (id, class, href, style, etc.) and in this instance, we are using it to set the CSS styles inline.

This paragraph will become red with white text.

Getting User Input and Using it

In this last example, we are going to use a form with inputs and a button to allow a user to submit their first and last name. After they submit their name, an alert will appear on the page and their name will be output on the page in a paragraph as well. You should always use labels in your forms for accessibility purposes. The for attribute value in the label tag must match the id attribute value in the input tag used.

For most examples, we have set the type attribute for each button to "submit." However, for this example, we use a value of "button" because the page will refresh after we submit our inputs. Once the inputs are submitted, the alert would appear along with the text but it would disappear shortly after. The button attribute value allows our content to stay on the page.
