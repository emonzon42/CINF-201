Basic Flexbox Example

On this page, I have created a few multi-column layouts which utilize the flexbox property. To do this, I give the parent container (row) a display of flex. I can then use margins/width on each of the children containers to style the as I'd like. The content here is structured similarly to what you'll find in the Floats Example page.

For any number of columns, you would take 100 and divide it by that number. If I want a 4 column layout, 100/4 would be 25%. This means I need at least 25% space for each set of columns. You could do something like give each column a width of 20% with 2.5% margins on each column (20 + (2.5*2) = 25).

In this example, I've created a 3-column layout with one flex wrapper and three divs inside of it. I used 30% for the width and 1.66% for the margins on each side of the divs.

The image in this example hasn't been floated. Instead, I gave the parent container (flex) a display of flex. That makes it so the image and p tags are placed horizontally. From there, we can add margins and padding to give it the same spacing as in the floats page.
