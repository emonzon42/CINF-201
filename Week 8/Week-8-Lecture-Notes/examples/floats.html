Basic Float Example

On this page, I have created a few multi-column layouts which utilize the float property. To do this, I targeted the row containers and their before/after pseudo-classes. I then floated the child divs to the left and gave them a width/margin.

For any number of columns, you would take 100 and divide it by that number. If I want a 4 column layout, 100/4 would be 25%. This means I need at least 25% space for each set of columns. You could do something like give each column a width of 20% with 2.5% margins on each column (20 + (2.5*2) = 25).

In this example, I've created a 3-column layout with one row wrapper and three divs inside of it. I used 30% for the width and 1.66% for the margins on each side of the divs.

The image to the left of this paragraph has been floated. Since it's floated, the text here will flow around it. Being that this content is in a div with a class of "row", it won't move up or down. I've added margin to the right of the image to give it space from the text. This div has been given padding to allow there to be space between the content and the edge of our container.
