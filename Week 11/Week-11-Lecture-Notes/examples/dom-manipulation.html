Selecting One Element Using its ID

In this example, we use the getElementById() method to select one element on the page and give it some text/change its background color to lightblue. That element must have a unique id value or our code won't work.

Selecting Elements Using a Class Name

In this example, we use the getElementsByClassName() method to select multiple elements at once. Unlike getElementById, this method gets a collection of elements based on their class name. Collections are similar to arrays and we can get their length, loop through them, and reference individual items by their index number.

After I get the elements as a collection, I loop through them to give each one a navy background and white text. I then target one item in that collection to give it a bigger font size.

Selecting Multiple Elements Using a CSS Selector

In this example, we will use the querySelectorAll method to select all elements which match the CSS selector we provide and then style them. When we use this method, it's very similar to document.getElementsByClassName. However, unlike the class name method, we can select multiple elements based on any CSS selector we provide instead of just the class name.
